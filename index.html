CareTracker Website

I developed a responsive web application that allows parents to register their children using HTML, CSS, and Bootstrap. Built with : HTML, CSS, Bootstrap , Javascript, php, Mysql, XAMPP, phpMyAdmin, GitHub

Weather Website :

Weather Website

The website features a clean and intuitive user interface built with HTML and styled with CSS and Bootstrap, ensuring a responsive design that adapts to different screen sizes. Users can input the name of a city into the search bar and click the search button. Using JavaScript, the web app interacts with the Weather API by API-Ninjas to retrieve the weather data for the specified city. The API provides comprehensive weather information including temperature, cloud percentage, humidity, wind speed, and more.
Built with : HTML, CSS, Bootstrap, Javascript, Weather API by API-Ninjas
